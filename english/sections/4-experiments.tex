\section{Experiments}  \label{Experiments}

We conduct two experiments on one for frame dropout and sensor drop out. First we analyze model performance in scenario when single sensor input is degraded by dropping its frames. Second, we analyze model performance multi-sensor scenario when some of the sensors experience failure.

\subsection{Frame Dropout}

We ran a "default" evaluation where none of the frames were dropped. We evaluated the performance separately for each different number of digits per video sequence. Table~\ref{tab:results_frame_dropout_default} shows the inference results. Both models show relatively similar results.
